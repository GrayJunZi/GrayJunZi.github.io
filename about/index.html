关于我

本人是一名 Web 开发工程师，以 系统架构师 为终极目标，学习并使用过的编程语言有 Java、.NET、Node.js 以及 Go 等。目前正在学习 微服务 架构知识。

从 2016 年开始来上海工作至今已有几年的时间, 感觉自己的技术和科班出身的还是有一定的差距。也没有朋友, 磕磕碰碰几年下来也没攒下什么技术, 什么都学什么都只学会一半笑称自己为 半水大师(意为自己只是个半桶水、半吊子的码农)。

从入职的第一家物流公司开始, 就已经离当初定下的目标越走越远了。刚入职的时候发现公司只有我一个做开发的, 而我当时也是个菜鸟加上年龄比较小没有什么社会经验。

工作第一年的时候是走的 Java 方向 用的 SSM(Spring SpringNVC MyBatis)技术来开发项目, 而我在学校的时候学的是SSH(Spring Structs2 Hibernate)没用过SSM， 不过我自学了几天就学会了, 可能当时自己求知欲太强烈的缘故吧。

到了第二年的时候项目使用的是 WebForms 技术, 我也就转而做.NET开发了, 到了中旬的时候开始招人,领导让我面试和带领团队, 之后我就带了四五个应届毕业生又开发了一年的项目, 管理对于刚出社会没多久的我来说着实是个挑战了。但后来因为觉得自己所掌握的技术都已经过时了, 为了能够让自己能够实现自己定下的目标, 2018年初就离职寻找下份工作了。

  • 我其实是感激第一家公司的毕竟是他们把我从哈尔滨带到上海的, 但不代表我欠他们什么, 我的一切都是靠自己的努力得来的。
  • 第一家公司对我来讲可以说是恨又无助, 一方面是他们觉得我年龄小好欺负。另一方面是自己性格问题比较老实也没有涨过工资。
  • 所有人都在否定我, 因为自己的硬性缺点: 年龄、学历、技术、社交能力。
  • 我现在依旧觉得自己技术很烂, 但是我没有能力提升, 学的越多越觉得自己会的太少。
  • 我浮躁、我焦虑、我恐慌, 我怕自己找不到工作, 我怕得不到提升, 我怕实现不了自己的理想。

2018年5月历时两个多月终于找到了一份满意的工作, 是一家做医疗软件的公司但是驻场开发, 刚进去的时候环境很差后来工作两年也就习惯了。但自己的技术增长还是太慢, 在公司没有学到什么东西还是做着内部管理软件, 技术是 ASP.NET MVC。虽然自己平时自学了 .NET Core 但没有实战过, 像是 redis mongodb docker 这些常用技术自己都没有用使用过, 同事们也不关心技术, 令我依旧感到孤独。

因为2017年的时候申请了专升本, 所以自己的平时又要顾虑学校考试, 又要自己平时提升自己, 事情太多了也就没换工作, 但始于技术终于技术, 依旧没有能和自己交流的技术的人, 我也不知道自己哪里不足, 不知道自己是强项, 过着走一步看一步的生活, 依旧焦虑、依旧恐慌、依旧迷茫、依旧手足无措。

2020 年初因为疫情, 公司开始走下坡路 2019年的下半年换了老板之后, 不专心搞业务核心而开始转向了严格管理员工, 开始趁火打劫弄竞业协议与培训考试的事情, 自己因为这些事情也烦躁的不行而自己的技术也没有且又是两年没有涨工资, 四月申请离职, 离职交接期间投的简历都石沉大海没有回应, 自己 依旧焦虑、依旧恐慌、依旧迷茫、依旧手足无措。

之前的领导去了前老板开的新公司去了, 因为觉得我的技术不错, 希望把我也带过去, 公司是初创公司忍受进去, 加班很严重, 当时自己的想法是不想去觉得肯定对自己又是没有提升的地方, 想着离职之后休息休息去个其他城市旅游放松一下然后闭关修炼把知识恶补一下去一家自己想去的公司, 领导又来找我希望我去新公司的时候, 我把薪资调高了一些, 以为疫情期间而自己技术没有那么厉害以为不会同意的, 没想到领导还是同意了然后盛情难却自己好面子也不好意思说不去, 五一假期结束之后就去那个新公司办理的入职。

加班很严重, 领导技术确实不错, 但是自己的水平太低了, 有时希望能提供一些思路说的很抽象, 自己不理解,只好硬着头皮做了。项目是做的SaaS系统 技术是 WebAPI、Redis、MongoDB，我也不知道自己下一步怎么该怎么做了呀，我也很累啊加班加的身心俱疲, 自己不是很能吃苦 有时想着别人也那么辛苦 觉得自己很没用啊, 依旧焦虑、依旧恐慌、依旧迷茫、依旧手足无措。

以下为本人需要恶补的技术：

  • 前端：Vue、React、Angular
  • 后端：Java、.NET、Go、Python、Node.js、Deno.js
  • 移动端：Flutter、React Native、uni-app、微信小程序
  • 即时通讯：WebSocket、WebRTC
  • 消息队列：RabbitMQ
  • 音视频处理：ffmpeg
  • 缓存数据库：Redis、Mongodb
  • 桌面应用程序：Electron、QT
  • 反向代理服务器：Nginx

理想

  • 自己开发一款非常成功的软件
  • 入职一家大型企业实现个人价值
