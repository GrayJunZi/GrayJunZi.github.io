Vue 项目基本结构

  • node_modules - 项目依赖的包所在位置
  • public - 公共资源文件
    • index.html - 运行及打包时的模板文件
  • src - 项目主目录
    • assets - 静态资源文件
    • components - 组件文件
    • views - 视图文件
    • App.vue - 基础组件
    • index.js - 项目入口文件
    • router - 路由文件
    • store - Vuex 状态管理文件
  • .browserslistrc
  • .editorconfig
  • .eslintrc.js - eslint 语法规则配置文件
  • babel.config.js -
  • package-lock.json - 锁定依赖包的版本
  • package.json - 定义了项目描述，包含项目名称、版本、及依赖的包
    • name - 项目名称
    • version - 项目版本
    • private -
    • scripts - 可执行的脚本
    • dependencies - 项目打包之后引用的依赖
    • devDependencies - 项目开发时引用的依赖
  • vue.config.js - vue 配置文件
    • lintOnSave - true 或 false 来处理是否保存时进行 eslint 语法检测。
    • chainWebpack - 配置 webpack
    • productionSourceMap - 打包时不产生 map 文件，降低文件体积，加快打包速度
    • devServer - 解决跨域问题

扩展目录结构

  • src
    • api - 存放项目接口的文件夹
    • config - 存放项目配置的文件夹
    • directive - 存放 vue 自定义指令的文件夹
    • lib - 存放基础设施的文件夹
    • mock - 存放模拟数据的文件夹
      • mockjs - 在开发者模式中安装 mockjs 依赖包npm install mockjs -D
