Go 入门指南

Go语言也称 Golang 是由 Google 公司开发并开源的编程语言, 它将简洁发挥到极致, Go语言之父 Rob Pike 设计之初以 C 为原型, 以 C++ 为目标设计的, 尽管最终大相径庭但值得一提的是, Go如其名最大的优势在于快, 开发快编译快运行快, 而且Go的库和生态也不错, 社区很活跃。

Go 语言具有很强的表达能力，它简洁、清晰而高效。得益于其并发机制， 用它编写的程序能够非常有效地利用多核与联网的计算机，其新颖的类型系统则使程序结构变得灵活而模块化。 Go 代码编译成机器码不仅非常迅速，还具有方便的垃圾收集机制和强大的运行时反射机制。 它是一个快速的、静态类型的编译型语言，感觉却像动态类型的解释型语言。

本篇为 Go 语言的快速入门, 通过一些简单示例带你熟悉这门简单易学的编程语言。

一、Go语言的前景？

Go语言在近两年的发展速度非常快, 一方面 Go语言 天生支持并发, 并且拥有非常优秀的语言设计足够简单灵活且自带完善的工具链，而且部署简单方便。加上Go语言的语法特性简单, 学习曲线很低, 易上手,开发效率高、运行效率快。

Go语言在全球范围内都有大量的优秀成功案例, 首当其冲就是 Docker 容器虚拟化技术, 在 Web打包及云计算领域有着举足轻重的地位。还有以下等项目都是采用Go进行开发的。

  • Nsq：bitly开元的消息队列系统，性能非常高，目前他们每天处理数十亿条消息。
  • Kubernetes：容器编排工具，实现自动化部署，更新，下线，负载均衡，容错处理等。
  • Fabric：区块链超级账本Hyperledger Fabric实现，用于联盟链开发。
  • Syncthing：一个持续不断的文件同步项目，能够在两台或者多台电脑上同步文件，使用了其独有的对等自由块交换协议，速度极快！
  • Lantern：中文名“蓝灯”。分布式，点对点的自由上网解决方案。你懂得！

Go语言发展至今已有无数拥簇者, Google 作为开发Go语言的公司自然不用说,当仁不让开发了许多基于Go的优秀项目例如Kubernates。以及国外社交网络服务巨头Facebook公司也是Go的忠实簇拥者开发了grace平滑重启工具。腾讯作为国内三巨头之一, 在2015年就已经做了万台规模的docker实践, 如今也有大量的Go职位在招聘, 七牛云作为国内多媒体存储巨头, 承接大量中小型企业的图片、视频等多媒体文件,其用到的技术栈就是Go, CEO许式伟更是Go的拥簇者还写了本《Go语言编程》、还有小米 等更多大厂都在使用 Go 来开发项目, 可见Go的未来一片光明。

Go语言在Web开发中也有许多的框架可用，例如以下：

  • Beego - Beego 是 astaxie 开源的一个用于开发Web的框架, 作者借鉴了PHP框架的思想。
  • Gin - Gin是一个golang的微框架, 封装比较优雅, API友好, 源码注释比较明确, 其具有快速灵活容错方便等特点被越来越多的开发者青睐, 目前GitHub的star数已超36k
  • Iris - Iris框架在其官网上被描述为Go开发中最快的Web框架。
  • GoFrame - 是由国人开发,是一个功能齐全的具有模块化、高性能、生产级等诸多优点的开发框架, 官方文档详细中文支持好, 学习曲线最低。
  • Yoyogo - 是基于ASP.NET Core设计的Golang实现, 是一个用 Go 编写的简单，轻便，快速的 微服务框架，目前已实现了Web框架的能力，但是底层设计已支持多种服务架构。

三、基础语法

1、输出语句

  • 在输出语句的字符串中可使用转义字符: \n 换行, \t 制表符。
  • 格式化输出: %s 输出字符串, %d 输出整型, %f 输出浮点型, %T 输出数据类型

2、变量

变量(variable), 指的是在程序运行过程中可以被改变的值的标识。

  • 在程序运行过程中不可改变的叫做 常量
  • 在 Go 语言中一行只有一条语句的时候可以不用写分号, 编译器会在执行的时候自动在最后加上分号,如果一行有多条语句则必须要加上分号。
