linux/linux-basic

Linux

基本语法

1. 查询目录

  • ll - 查询目录（管理员才能使用）
  • ls - 查询目录
  • ls -al - 与 ls 查询内容相同

2. 目录结构

  • bin - 存放二进制可执行文件（ls, cat, mkdir 等命令文件）
  • boot - 存放用于系统引导时使用的各种启动文件
  • dev - 用于存放设备文件
  • etc - 存放系统配置文件
  • home - 存放所有用户文件的根目录
  • mnt - 系统管理员安装临时文件系统的安装点
  • opt - 额外安装的可选应用程序包所存在的位置
  • proc - 虚拟文件系统，存放当前内存的映射
  • root - 超级用户目录
  • sbin - 存放二进制可执行文件，只有 root 才能访问
  • tmp - 用于存放各种临时文件
  • usr - 用于存放系统应用程序，比较重要的目录/usr/local 本地管理员软件安装目录
  • var - 用于存放运行时需要改变数据的文件
