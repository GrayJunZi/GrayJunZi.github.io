Linux 目录说明

  • bin - 存放二进制可执行文件
  • boot - 存放系统引导时使用的各种文件
  • dev - 用于存放设备文件
  • etc - 存放系统配置文件
  • home - 存放所有用户文件的根目录
  • lib - 存放根文件系统中的程序运行所需的共享库及内核模块
  • mnt - 系统管理员安装临时文件系统的安装点
  • opt - 额外安装的可选应用程序包所放置的位置
  • proc - 虚拟文件系统，存放当前内存的映射
  • root - 超级用户目录
  • sbin - 存放二进制可执行文件，只有 root 才能访问
  • tmp - 存放各种临时文件
  • usr - 存放系统应用程序，比较重要的目录 /usr/local 本地管理员软件安装目录
  • var - 存放运行时需要改变数据的文件

IP

  • ip a - 查询 IP 地址
  • ifconfig - 查看 IP

目录操作

  • ll - 文件列表（仅管理员用户可用）
  • ls - 显示文件和目录列表
  • ls -al - 显示当前目录的文件列表
  • pwd - 查询当前所在目录
  • cd / - 切换到根目录
  • cd .. - 切换到上级目录
  • mkdir - 创建目录
  • touch test.txt - 创建空文件
  • echo hello > 1.txt - 创建或重写自定义内容的文件
  • echo word >> 1.txt - 在文件中追加自定义内容
  • cat 1.txt - 查看文本文件内容
  • cp 1.txt 2.txt - 拷贝文件
  • rm 2.txt - 删除文件
  • mv 1.txt .. - 剪切文件，移动文件位置到上级目录
  • find . -name test.txt - 查找文件，从当前目录中查找名为 test.txt 的文件
  • grep - 在指定的文本文件中查找指定的字符串，
  • cat 1.txt | grep hello - 在 1.txt 文件中查找 hello 字符串并显示出来
  • tree - 以树状图列出目录结构（需安装软件）
  • ln 1.txt 2.txt - 创建软链接，类似快捷方式
  • more .bashrc - 分页显示文本文件内容（ctrl + c 中断命令）
  • head .bashrc - 显示文件开头内容
  • tail .bashrc - 显示文件结尾内容
  • ping - 心跳检测
  • sudo - 提权，即提升权限
  • rm -fr * - 删除当前目录下的所有文件
  • ctrl + c - 中断命令
  • ctrl + d - 退出用户

压缩操作

  • tar - 压缩与解压缩命令

  • tar -c - 建立一个归档文件，即压缩命令

  • tar -x - 解压一个归档文件，即解压命令

  • tar -z - 是否需要用 gzip 进行压缩

  • tar -j - 是否需要用 bzip2 进行压缩

  • tar -v - 压缩过程中显示文件

  • tar -f - 使用归档名称，在 f 之后要立即接档名

  • tar -tf - 查看归档文件里面的文件

  • tar -czvf test.tar.gz . - 将当前目录的所有文件以 gzip 压缩算法进行压缩到 test.tar.gz 文件中，并显示

  • tar -zxvf test.tar.gz - 将 test.tar.gz 文件以 gzip 压缩算法进行解压至当前目录中，并显示

系统管理命令

  • stat - 显示指定文件的相关信息，比 ls 命令显示的内容更多
  • who - 显示在线登录用户
  • hostname - 显示主机名称
  • uname - 显示系统信息
  • top - 显示当前系统中耗费资源最多的进程，即任务管理器
  • ps - 显示瞬间的进程状态
  • du - 显示指定文件或目录已使用的磁盘空间的总量
  • df - 显示文件系统磁盘空间的使用情况
  • df -f - 显示文件系统磁盘空间的使用情况（数值换算）
  • free - 显示当前内存的交换空间的使用情况
  • ifconfig - 显示网络接口信息
  • ping - 心跳检测
  • netstat - 显示网络状态信息
  • clear - 清屏
  • kill - 杀死一个进程
  • sudo reboot - 重启
  • sudo shutdown -r now - 立即重启
  • shutdown -h now - 立即关机

Vim 编辑器

  • 运行模式
    • 编辑模式 - 等待编辑器输入
    • 插入模式 - 编辑模式下，输入 i 进入插入模式，插入文本信息
    • 命令模式 - 在编辑模式下，输入 : 进入命令模式
  • 命令模式
    • vi 1.txt - 打开文本编辑器并编辑 1.txt 文件
    • :q - 退出 vim
    • :q! - 强制退出 vim
    • :w file - 将当前文件内容保存为某个文件
    • :wq - 保存并退出 vim，可新建文件
    • :wq! - 强制写入并退出
    • / - 查找字符串
    • set number - 在 vim 编辑器中显示行号
    • set nonumber - 在 vim 编辑器中不显示行号
    • set paste - 原样粘贴
    • esc - 退出命令模式快捷键进入编辑模式

Linux 用户与组管理

  • sudo passwd root - 修改用户密码

  • su - 切换到 ROOT 用户

  • 设置允许远程登录 root

文件权限管理

  • - - 无权限

  • r - read 可读

  • w - write 可写

  • x - execute 可执行

  • - 普通文件

  • rw- 用户 有读写权限，没有运行权限

  • r-- 用户组 只有读权限，没有写和运行的权限

  • r-- 其他用户只有读权限，没有写权限和运行的权限

  • drwxr-xr-x - 文件权限

    • d 文件类型（d:directory）
    • rwx - 用户权限
    • r-x - 用户所在组权限
    • r-x - 其他用户权限

  • chmod - 赋予权限

    • chmod +x test.sh - 为 test.sh 文件赋予可执行权限
    • chmod -x test.sh - 为 test.sh 文件删除可执行权限

  • chown - 切换组

    • chown test:test test.sh 为 test.sh 文件切换组

  • 数字表示法

    • 0 无
    • 1 执行
    • 2 写
    • 4 读
    • chmod 706

Linux 软件包管理

  • lsb_release -a - 查看系统版本

  • 修改软件数据源

  • apt-get update - 更新数据源
  • apt-get install packagename - 安装软件包
  • apt-get remove packagename - 删除软件包
  • apt-get upgrade - 升级有可用更新的系统
  • apt-get clean && apt-get autoclean - 清理无用的包
  • apt-get check - 检查是否有损坏依赖
